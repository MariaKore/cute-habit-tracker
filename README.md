Title: Cute habit Tracker v1
A habit tracker with a very cute UI :)

For this project, I followed this youtube tutorial: https://youtu.be/-OcVVgDLI3U?si=Oe9Dz9ur2uIogBQz
Specs:
- A monthly tracker, set to your current month
- Tracks day by day; you can see all the days of the current month
- Easy click and un-click feature
- Large reset button for readibility and when you need to clear the tracker instantly
- Lots of pink colors since pink is the best color ^_^
